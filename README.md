# 遊戯王ジェネシスポイントビューワー

Yu-Gi-Oh! GENESYSのポイントを、遊戯王ニューロンのデッキ詳細画面に表示・集計するChrome拡張機能です。

## 概要

この拡張機能は、デッキリストに含まれるカードのGenesysポイントを自動で検出し、カード名横にバッジで表示するとともに、デッキの合計ポイントを計算し、デッキ枚数表示エリアに追加表示します。

## 機能

* **カード別ポイント表示:** デッキリスト内の各カード名の横に設定されたポイントをバッジで表示します。
* **合計ポイントの計算:** デッキリストに記載された**枚数**に基づき、デッキ全体の合計ポイントを計算し表示します。
* **表示形式対応:** 「カード詳細表示」および「テキスト表示」の両方のデッキ表示形式に対応しています。

---

## ファイル構成

-	manifest.json: 拡張機能の設定ファイル
-	content.js: Webページを操作するスクリプト (ロジック本体) 
-	styles.css: ポイント表示のデザイン 
-	points.json: カード名とポイントの対応表

## 🛠️ インストール方法

本拡張機能は、Chromeの「デベロッパーモード」機能を使用してローカルで読み込む必要があります。

### 1. リポジトリのダウンロード

まず、このGitHubリポジトリ全体をローカル環境にダウンロードします。

1.  GitHubリポジトリページ右上の **Code** ボタンをクリックします。
2.  **Download ZIP** を選択し、ファイルをダウンロードします。
3.  ダウンロードしたZIPファイルを解凍し、拡張機能のフォルダを用意します。

### 2. Chromeへの読み込み手順

1.  Chromeブラウザを開き、アドレスバーに `chrome://extensions` と入力して拡張機能管理ページを開きます。
2.  画面右上の **デベロッパーモード** のトグルをオンにします。
3.  画面左上の **パッケージ化されていない拡張機能を読み込む** ボタンをクリックします。
4.  手順1で解凍したフォルダを選択します。

読み込みが成功すると、拡張機能リストに「Yugioh Genesys Point Viewer」が追加されます。

## 利用方法

拡張機能の読み込み完了後、遊戯王ニューロンのデッキ詳細画面を開くか、既に開いている場合はページをリロードしてください。

* `points.json` に設定されているカード名にポイントバッジが表示されます。
* デッキ枚数表示エリアに「ポイント合計」が追加表示されます。

## 過去のリミットについて
過去のリミットを適用したい場合は、本リポジトリの [data-archives](../../tree/data-archives) ブランチに移動し、`data_history/` 内のファイルを `points.json` に上書きコピーして拡張機能を再読み込みしてください。
